Documentação HTML

As tags semânticas nos ajudam a estruturar o conteúdo da nossa página de uma forma lógica e significativa, permitindo que leitores de tela, por exemplo, possam identificar facilmente o que é um título, um parágrafo ou uma lista.

A documentação oficial pode ser vista no site w3schools

Textos

<p> </p>

Tag HTML responsável por definir o cabeçalho ou título de uma página
<h2> a <h6>
Tag HTML responsável por definir o subcabeçalho ou subtítulo de uma página

Quebra de linha
<strong> </strong>
Negrito
<em> </em>
Itálico ou Ênfase
<span> </span>
É um conteiner generico em linha para conteúdo fraseado, que não representa nada por natureza. Ele pode ser usado para agrupar elementos para fins de estilo (usando os atributos class ou id )
<abbr title=""></abbr>
Abreviação, onde em title pode ser colocado o significado da abreviação usada, como aqui em HTML

Imagens

<img src="caminho ou url" alt="texto alternativo">

A tag img é responsável por adicionar imagens a uma página, esta tag tem 2 propriedades.

src="" que recebe o caminho da imagem, podendo ser um caminho para um diretório local ou URL para web, este caminho deve também conter a extenção do arquivo

alt="" que recebe um texto alternativo para a imagem caso ele não apareça, importante caso algum problema ocorra

link:favicon

A tag link:favicon é responsável pela adição de um ícone na aba do site, este ícone deve ser do tipo .ico

É aconselhavel que o nome do ícone seja favicon

Áudio e Vídeo

Áudio


    <audio controls loop preload="metadata">
        <source src="midia/audio/tom-petty-love-is-a-long-road.mp3" type="audio/mpeg">
    </audio>

<audio src="midia/audio/tom-petty-love-is-a-long-road.mp3" controls loop preload="metadata" ></audio>

controls é necessário para que o client consiga controlar a reprodução do áudio

preload é usado para carregar algumas informações importantes sobre o conteúdo

loop mantem o áudio si repetindo sempre que acabar a reprodução

src é necessário para inserirmos o caminho para o conteúdo que queremos reproduzir

Vídeo Hospedado Localmente

<video src="midia/video/meu-video.mp4" controls loop preload="metadata" ></video>

controls é necessário para que o client consiga controlar a reprodução do video

preload é usado para carregar algumas informações importantes sobre o conteúdo

loop mantem o video si repetindo sempre que acabar a reprodução

src é necessário para inserirmos o caminho para o conteúdo que queremos reproduzir

Vídeo Hospedado externamente

Quando o vídeo está em uma hospedagem externa, o proprio site já ofecere o código necessário para linkar o vídeo ao site

Link's ou Âncoras

<a href="URL" target="" rel="" download="" type=""> texto ou imagem </a>

O trecho href="" recebe a URL para o qual o link aponta

O trecho target="" recebe um valor que pode variar entre _blank ou _selft ( padrão), com o valor _blank a página será aberta em uma nova guia, já com o valor padrão _self a página será aberta na guia atual

O trecho rel="" indicará ao motor de busca que o link é para outro site ou se é para um site de mesmo autor, ou até mesmo para uma página anterior ou adiante do mesmo documento, ou seja, se é external, author, prev, next

Este trecho download="" indicará o caminho e nome do arquivo para download

Este trecho type="" indicará o caminho e nome do arquivo para download, o tipo de arquivo deve ser consultado em iana.org

Diferença entre id e class

ID

O identificador ID é utilizado para identificar tags, dessa forma, uma estílização pode ser feita específicamente para aquela tag HTML

Se mais de uma tag receber a mesma ID, apesar de funcionar normalmente, não é indicado como correto pelo w3schools, para estes casos é melhor utilizar classes

Uma outra característica importante do ID é que sua hierarquia é superior a da Class, ou sejá, se duas estilizações estiverem afetando um elemento da mesma forma, aquele que irá sobrepor é o que está contido no ID

                    HTML
<p id="paragrafo">
    texto
</p>
                    
                
                    CSS
#paragrafo{
    background-color: #e5e5e5;
    width: 300px;
}

Class

O identificador class é usado para identificar e estilizar uma ou mais tags em uma página

É importante não confundir a forma de utilizar o ID e Class no CSS, visto que o ID usa # e o Class usa um ponto

                HTML
<p class="paragrafo">
    texto
</p>
            
            
                    CSS
.paragrafo{
    background-color: #e5e5e5;
    width: 300px;
}

Pseudo-Classes e Pseudo-Elementos

Uma pseudo-classe e pseudo-elementos CSS é uma palavra-chave adicionada a seletores que especifica um estado especial do elemento selecionado.

Por exemplo, :hover pode ser usado para alterar a cor de um botão quando o usuário passar o cursor sobre ele.

Pseudo-Classes

                    CSS
.little_code:hover{
    background-color: #bebebe;
}

Pseudo-Elementos

                    CSS
a::after{
    content: " [link]";
}

ShortHands

As ShortHands são propriedades abreviativas no CSS que deixam você setar os valores de outras propriedades de CSS simultaneamente

                    CSS
div{
    padding: 10px 15px 20px 50px;
    /*
    padding-top: 10px;
    padding-right: 15px;
    padding-bottom: 20px;
    padding-left: 50px;
    */
}
