Iframes

Exibir códigos sem o uso de Iframes

Temos algumas formas de fazer isso, umas delas é através do uso de algumas tags específicas, como a tag <span>, <code> e <pre>. O resultado abaixo é um código sem o uso da tag <pre> que tem como objetivo manter a pre formatação, como identação, como resultado temos um código bagunçado

<code> <ul> <li> item 1 </li> <li> item 2 </li> </ul> </code>

Agora utilizando a tag <pre>

<pre>
    <code>
        <ul>
            <li>
                item 1
            </li>
            <li>
                item 2
            </li>
        </ul>
    </code>
</pre>

Além disso algumas alterações podem ser feitas, como adição de uma div, com shadow box interno, padding e cor de fundo

    padding: 5px;
    box-shadow: inset 0px 0px 3px 0px rgba(0, 0, 0, 0.500);

Exibir códigos com o uso de Iframes

Aqui estou usando um iframe para abrir um arquivo de texto que contém o código, a desvantagem disso é não pode alterar as características visuais, como font, cor da letra e do fundo.

Neste segundo exemplo eu abri um arquivo .html externo que contém o código, dessa forma o iframe interpreta como um site e por isso ele é guiado pelas configurações de estilo dessa página
